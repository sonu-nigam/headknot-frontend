# Product screenshots

Drop the marketing screenshots here. The `ProductShowcase` section on the
landing page loads them by filename and shows a branded placeholder until the
real image exists.

Expected files (PNG or JPG, ~16:10, ideally ≥1600px wide):

| File                     | View                |
| ------------------------ | ------------------- |
| `knowledge-graph.png`    | Knowledge Graph View |
| `search-results.png`     | Search Results View  |
| `entity-details.png`     | Entity Details View  |

If you change a filename, update `src` in
`apps/website/src/components/ProductShowcase.tsx`.
